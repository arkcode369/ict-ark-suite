# QT — Struktur Dasar

## Q1–Q4 AMD Cycle

```
Q1 → Q2 → Q3 → Q4

A = Accumulation    (range / konsolidasi)
M = Manipulation    (false break / sweep liquidity)
D = Distribution    (expansion / delivery)
X = Continuation or Reversal

Dua profil utama:
  XAMD  →  Q1 sudah Expansion → Q2 Accumulate → Q3 Manipulate → Q4 Distribute
  AMDX  →  Q1 Accumulate → Q2 Manipulate → Q3 Distribute → Q4 Expansion/Reversal

Key rule:
  → Q1 dictates the entire cycle
  → If Q1 = X (Expansion) then Q2 should be A (Accumulation)
  → Trade hanya di M (Manipulation) atau D (Distribution)
  → Every Q2 open = "True Open" — reference level untuk seluruh cycle

Q4 Key:
  → Jika Q4 over-extended dari Q2 → expect REVERSAL
  → Jika Q4 NOT over-extended dari Q2 → expect CONTINUATION
  → 4th week of month (Q4 Monthly Cycle) = biasanya lack-luster
    karena price sudah mencapai liquidity objective
```

# Lesson 14 — sSMT (Sequential SMT)

## Definisi

```
sSMT = SMT yang terjadi dengan:
  → Highest High ATAU Lowest Low dari QUARTER SEBELUMNYA

Regular SMT: SMT antara dua aset pada swing apapun
sSMT:        SMT yang secara spesifik menyentuh high/low dari Q sebelumnya
             → Lebih signifikan karena mengambil liquidity di boundary Q sebelumnya
```

## 3 Bentuk sSMT

```
1. Normal sSMT (FS = First Stage):
   Quarter A  |  Quarter B
   SMT dengan Highest High / Lowest Low dari Q sebelumnya
   → Standard form, cukup valid

2. Hidden sSMT (HSMT):
   Quarter A  |  Quarter B
   SMT dengan Highest BODY / Lowest BODY dari Q sebelumnya (bukan wick)
   → Lebih tersembunyi, menggunakan candle body sebagai referensi

3. Closure sSMT (TS = Third Stage / Paling Kuat):
   Quarter A  |  Quarter B
   Kombinasi Normal sSMT + Hidden sSMT dalam satu swing
   → HIGHEST probability dari ketiga bentuk
   → Both wick dan body memberikan sSMT signal sekaligus
```

## 2-Stage sSMT

```
2 Stage sSMT:
  FS (First Stage)  = HTF cycle sSMT (misal Daily Cycle)
  TS (Second Stage) = LTF cycle sSMT (misal Session Cycle) → confirms expansion

Sequence:
  1. HTF sSMT terbentuk (First Stage konfirmasi)
  2. LTF sSMT terbentuk (Second Stage konfirmasi)
  3. Expansion dimulai setelah Second Stage

"2 Stage sSMT involves going from HTF cycle's sSMT to LTF cycle's sSMT to confirm expansion"
```

## Kapan sSMT Gagal?

```
sSMT kemungkinan GAGAL jika:
  1. Bullish sSMT ABOVE True Open dari cycle tersebut → salah sisi
  2. Bearish sSMT BELOW True Open dari cycle tersebut → salah sisi
  3. HTF orderflow berlawanan dengan arah sSMT
  4. Tidak ada karakteristik manipulasi pada aset yang trigger second stage:
     → Tidak ada CSD
     → Tidak ada OB
     → Tidak ada iFVG (Inverse FVG)
     → Tidak ada Breaker
```

# Lesson 21 — Magneto Theory + Doubling Theory

## Magneto Theory

```
Definisi:
  Magneto = old SSMT highs/lows berfungsi sebagai TARGET

Setup Magneto:
  1. Temukan old SSMT low/high (dari swing sebelumnya)
  2. New SSMT terbentuk
  3. CSD dikonfirmasi menuju old SSMT level
  4. Target = old SSMT level (seperti magnet menarik harga kembali)

Logic:
  Old SSMT levels = area liquidity penting yang belum diselesaikan
  → Price sangat sering "kembali" ke level-level ini
```

## Doubling Theory

```
Kapan Magneto menjadi Doubling Theory?
  → Ketika SATU ASET tidak menyelesaikan Magneto (tidak mencapai old SSMT level)
  → Membuat crack in correlation menjadi dua kali lipat
  → Probabilitas lebih tinggi dari Magneto biasa

Bentuk lain Doubling Theory:
  → SMT antara DUA BULAN yang berbeda (bukan quarters)
  → Karena months ≠ quarters → SMT antara mereka = Doubling Theory
  → Sangat high probability karena crosses cycle boundaries

IMSSMT + SSMT = Doubling:
  → Ketika IMSSMT terjadi bersamaan dengan SSMT dalam satu swing
  → Crack in correlation menjadi two-fold
  → Konfirmasi paling kuat

"When IMSSMT occurs in the presence of SSMT it is considered doubling as the crack in correlation is two fold."
```

# Mismatched Cracks in Correlation

```
4 tipe candle yang membentuk SMT berdasarkan struktur wick/body:

1. Body No Wick:
   Satu aset: candle yang close-nya kuat (full body, tanpa wick di ujung)
   Aset lain: membentuk wick (penetration)
   → Sering menghasilkan reversal yang lebih kuat

2. No Wick No Body:
   Satu aset: doji atau inside bar (tanpa body signifikan, tanpa wick)
   → Biasanya menandakan ketidakpastian

3. Wick No Body:
   Satu aset: wick panjang tanpa body (shooting star / hammer)
   Aset lain: close berbeda arah
   → Classic manipulation candle

4. Body Body Wick:
   Kombinasi: full body dengan wick
   Aset lain: berbeda arah
   → Menunjukkan decisive move dengan sedikit resistance

Mismatched = ketika dua aset menunjukkan TIPE BERBEDA dari kategori di atas
→ Mismatched Crack = reversal lebih kuat karena divergensinya lebih dalam
```

# Ringkasan Workflow QT

```
Step 1: Tentukan Quarterly Cycle Bias
  → Profil XAMD atau AMDX?
  → Kita sedang di Q berapa?
  → Trade hanya di M atau D quarter

Step 2: Tentukan True Open
  → True Open cycle yang relevan (Day/Session/Week)
  → ABOVE = bearish bias, BELOW = bullish bias

Step 3: Temukan DFR (jika di awal cycle)
  → DFR High, Low, EQ
  → Standard Deviations sebagai target

Step 4: Cari sSMT
  → Normal, Hidden, atau Closure sSMT?
  → Di boundary Q sebelumnya?
  → Apakah valid (tidak di sisi salah True Open)?

Step 5: Konfirmasi CSD
  → Harus ada SMT untuk CSD valid
  → Cek karakteristik manipulasi (OB, FVG, Breaker)

Step 6: Entry di LTF
  → 2-Stage: HTF sSMT → LTF sSMT → expansion
  → Precision Level: SMT/SSMT + Key Level + True Open + LTF Entry

Step 7: Target
  → Magneto (old SSMT levels)
  → DFR Standard Deviations
  → PSP/PC levels
  → NWOG dynamic targets

Step 8: Stop Loss
  → Bullish: Q2 Lows
  → Bearish: Q2 Highs
```
