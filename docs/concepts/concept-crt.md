# CRT — Core Concept

## Definisi

**Candle Range Theory (CRT)** = candle yang terbentuk di **key times** diperlakukan sebagai range.
Range tersebut bisa:
- **Truly broken** (breakout valid) → support untuk ekspansi lebih jauh
- **Falsely broken** (turtle soup) → ekspansi ke arah berlawanan

```
CRT vs Typical Range:
  Typical Range  = identified by H/L, subjective
  CRT Range      = fixed by open/close times, objective, more mechanical
  CRT advantage  = better alignment with top-down analysis
```

---

## Tipe CRT

```
1. 3-Candle CRT (Best / Most Reliable):
   Candle 1 = Range Candle      → establishes the range framework
   Candle 2 = Manipulation Candle → manipulates range (turtle soup)
   Candle 3 = Distribution Candle → expands toward opposing end

2. Inside CRT:
   Single candle = the range
   Multiple candles = manipulation & distribution phases
```

---

## 3-Candle Framework Detail

```
Candle 1 — Range Candle:
  → Seluruh price action candle ini = framework range
  → Tandai High dan Low dari candle ini

Candle 2 — Manipulation Candle:
  → Harga menembus range (baik ke atas atau bawah)
  → Sering membentuk Turtle Soup di sini
  → KRITIS: perhatikan close candle 2
    - Close INSIDE range  → High Probability Reversal (kuat)
    - Close OUTSIDE range → Lower Odds, butuh kehati-hatian

Candle 3 — Distribution Candle:
  → Target 1: 50% dari range (CE / Consequent Encroachment)
  → Target 2: Opposing end of range (sisi berlawanan)
```

---

## Trading Methods (Entry Types)

```
Method 1 — Orderblock Entry (2nd Candle):
  → Trading saat candle 2 terbentuk
  → Masuk setelah purge + displacement membentuk OB
  → Konfirmasi entry = moderately aggressive

Method 2 — Kiss of Death Entry (3rd Candle):
  → Trading saat candle 3 dimulai
  → Last turtle soup sebelum price mencapai opposing DOL
  → Lebih konservatif, entry di awal distribusi

Conformational CRT (Pemula):
  → Hanya trade candle ke-3, setelah full confirmation
  → Forget candle 1 & 2 — tunggu candle 3 mulai
  → Highest accuracy, tapi reward lebih kecil

Advanced CRT:
  → Trade candle 2 atau 3 sebelum konfirmasi penuh
  → Membutuhkan skill dan intuisi
```

---

## Di Mana Menemukan CRT

```
CRT terbentuk di:
  1. Range highs/lows
  2. Rejection of highs/lows (purge)
  3. HTF key levels (Daily, 4H)
  4. Turtle soups / purge area
  5. Manipulation phase
  → SELALU harus terbentuk di key times
```

## CRT + Time (Aturan Kritis)

```
  - CRT terbentuk di key time + purged di key time = TIME ALIGNMENT
  - Key purge times: 1AM, 3AM, 6AM, 8AM, 9AM, 11AM EST
  - LTF CRT dalam HTF CRT = highest priority setup
```

# Market Profiles (Weekly & Intraday)

## Weekly Profiles

```
1. Classic Expansion Week 1:
   → Bullish expansion from open, Monday anchor → week expands higher
   → SSL swept early week, price expands to BSL

2. Classic Expansion Week 2:
   → Similar structure with Monday manipulation then expansion

3. Consolidation Reversal:
   → Week starts consolidating, reverses mid-week
   → Low of week forms Tuesday/Wednesday

4. Midweek Reversal:
   → Expansion early week, reversal Wednesday/Thursday
   → High/Low of week typically formed Tuesday
```

---

## Intraday Profiles

```
1. Normal Protraction:
   → London leads, NY continues in same direction
   → HOD/LOD forms in London session

2. Delayed Protraction:
   → London accumulates (ranges), NY expands
   → HOD/LOD forms in NY session

3. NY Continuation:
   → London starts move, NY continues (no reversal)

4. NY Reversal 1:
   → London sets HOD, NY reverses and sets LOD (bearish reversal)

5. NY Reversal 2:
   → London sets LOD, NY reverses and sets HOD (bullish reversal)
```

> Key session marks: **LO** = London Open, **NYO** = New York Open

```
Basic Market Structure:
  → Highs & Lows only

Advanced Market Structure:

# Ch.10 & 11 — Range Trading

## 3 Tipe Range

```
1. Typical Ranges (Fixed H/L):
   → Range biasa dengan high dan low yang jelas
   → Identifikasi oleh swing H/L

2. CRT Ranges (HTF Candles):
   → Range dibentuk oleh candle HTF (Daily, 4H, Weekly)
   → Lebih objektif dan mekanis

3. Time Ranges:
   → Asian Range
   → CBDR (Central Bank Dealer's Range)
   → Monday Range
```

---

## Aturan Range Trading

```
1. JANGAN trade di DALAM range → terlalu banyak noise
2. Buy = saat harga breakout di atas range LOW (setelah sweep bawah)
3. Sell = saat harga breakout di bawah range HIGH (setelah sweep atas)
4. Target = 50% EQL (Equilibrium) dari range
5. Tunggu purge di key time sebelum entry
6. Kombinasikan dengan CRT untuk precision
```

---

## Turtle Soup dalam Range Trading

```
Bullish TS:
  → False breakout di BAWAH lows dalam bullish market (DOL higher)
  → Entry: setelah close kembali ke dalam range

Bearish TS:
  → False breakout di ATAS highs dalam bearish market (DOL lower)
  → Entry: setelah close kembali ke dalam range

High Probability TS Conditions:
  → Di atas/bawah CRT
  → HTF key levels
  → Key times (PALING PENTING)
  → PDH/PDL
  → Distribution phase
```

# Ch.12 — OHLC / OLHC

## OLHC — Bullish Candle

```
Open  = Range Phase       (accumulation, ranging)
Low   = Manipulation      (sweep SSL, Judas swing turun)
High  = Expansion Phase   (move ke BSL, DOL atas)
Close = Range / Reset

Liquidity Cycle:
  Generating → Purging → Neutralizing → Reset
```

## OHLC — Bearish Candle

```
Open  = Range Phase       (accumulation, ranging)
High  = Manipulation      (sweep BSL, Judas swing naik)
Low   = Expansion Phase   (move ke SSL, DOL bawah)
Close = Range / Reset
```

---

## Aturan Anticipatory OHLC

```
1. OPEN = Ranging Profile
   → Jangan trade langsung setelah open
   → Tunggu H/L terbentuk terlebih dahulu

2. H/L terbentuk di HTF Key Level = Manipulation
   → High/Low candle yang terbentuk di key level = manipulation signal

3. Expansion ke opposing HTF key level
   → Setelah manipulation, ekspektasi move ke sisi berlawanan

4. Close = Reset
   → Close candle kembali ke range = konfirmasi AMD selesai
```

# Ch.23 — Standard Deviations

## Definisi

```
The Mean:
  → "Fair value" / equilibrium di mana price accumulate orders
  → Market's perceived balance point
  → Contoh mean ranges: Asian Range, CBDR, Original Consolidation

Standard Deviations:
  → Mengukur seberapa jauh harga bergerak dari mean
  → Mengukur derajat volatilitas pasar
  → Highlight zones di mana price overstretched (potential reversal/expansion)
```

---

## Important STDv Levels

```
1 STDv     = Normal deviation
2 STDv     = Extended / potential reversal
2.5 STDv   = High extension
4 STDv     = Extreme extension
```

---

## Kegunaan Standard Deviations

```
1. Identifikasi area high-probability untuk reversal & expansion
2. Setting SL untuk Turtle Soup entries (PALING BAIK)
3. Mengukur protractionary / manipulation phase
4. Mengantisipasi HOD/LOD
5. Setting precise targets
```

---

## Integration dengan ICT

```
Requirements:
  → Harus diintegrasikan dengan key levels
  → Harus align dengan key times dan kill zones
  → Harus digunakan dalam arah draw on liquidity yang benar
```

---

## Cara Menggunakan STDv

```
Step 1: Identifikasi "the mean" (Asian range, CBDR, manipulation leg)
Step 2: Pilih correct range sebagai basis
Step 3: Plot key standard deviations dari mean tersebut
Step 4: Perhatikan HTF key levels — match dengan key STDv
Step 5: Tunggu aligned key level + key STDv ditest di key time
```

---

## STDv untuk Stop-Loss (Turtle Soup)

```
Rule:
  - Entry: di HTF key level yang align dengan STDv (misal: 2 STDv)
  - SL: di STDv berikutnya (misal: 2.5 STDv)
  - Logis: jika price melampaui 2.5 STDv → setup invalid

Anticipating HOD/LOD:
  - HOD/LOD hari biasanya terbentuk di 1–2 STDv Asian Range
  - Terkadang di 2.5 STDv
  - Match dengan key level + key time untuk presisi lebih tinggi
```

# Ch.24 — Mastering Turtle Soups (CRT Perspective)

## Definisi (ShamCRT)

**Turtle Soup** = false breakout / liquidity purge yang fokus pada liquidity hunt — market mentriger stop-loss retail sebelum berbalik. Dikoin oleh Linda Raschke. Merupakan entry method paling advanced.

```
Mengapa Efektif:
  → Capitalizes on retail mistakes
  → Leverages institutional moves
  → Clear risk-reward structure
```

---

## Karakteristik

```
Content:
  → Price moves beyond significant level briefly
  → Sudden reversal into prior range

Market Context (PENTING):
  → Terjadi dalam arah HTF draw on liquidity
  → Di sekitar HTF key levels
  → Saat liquidity hunts berlangsung
  → Di key times
```

---

## High Probability Turtle Soup

```
1. Clear & Defined Draw on Liquidity:
   → Harus accurately determined sebelum mencari TS

2. Sharp Narrative:
   → Weekly & intraday profiles, OHLC, PO3, dealing ranges

3. Market Context:
   → Key times (PALING PENTING)
   → Key levels (HTF)
   → Standard deviations (Fibonacci levels)
   → Sauce: best TS terjadi di 2, 2.5, atau 4 STDv dari mean ranges
```

---

## Refined Entries

```
Jangan:
  → Random entry setelah liquidity hunt
  → Entry ketika high/low baru saja terkena (tunggu move selesai)

Lakukan:
  → Gunakan key levels + STDv + Fibonacci + key times secara logis
  → Integrasikan semua elemen bersama
  → Adaptasi terhadap setiap skenario baru
```

---

## Stop-Loss untuk Turtle Soup

```
Pilihan SL:
  a. HTF key levels
  b. LTF refined key levels
  c. Standard deviations
  d. Orderblock stop-loss
  e. Swing points at key times
  f. Fixed pips (10, 15 pips, dll.)
  g. Kombinasi dari beberapa di atas (advanced)

Best Practice:
  → Pilih 2-3 confluences dari list di atas
  → Pelajari kombinasi yang works secara psikologis
  → Trust it consistently
```

---

## Step-by-Step Turtle Soup Approach

```
1. Start with Higher Timeframe
   → Identifikasi DOL menggunakan Daily dan 4H

2. Build a Narrative
   → Bagaimana price akan menuju DOL
   → Key level mana yang akan ditest dulu
   → Kapan liquidity akan diambil (key times)

3. Select Key Levels and Candle Ranges
   → Pilih key level + candle range yang relevan
   → Plot STDv dari mean ranges (Asian, CBDR, manipulation leg)

4. Wait for Liquidity Hunt / Manipulation
   → Di killzone atau key macro time

5. Refine Levels Using STDv
   → Match STDv dengan key level untuk setup lebih presisi

6. Set Stop Loss
   → Per SL protocol dari confluences yang dipilih
```

# Ch.26 — Swing Trading

## Definisi

Swing trading = menangkap pergerakan harga beberapa hari hingga beberapa minggu, memanfaatkan "swings" dalam orderflow yang lebih besar.

```
Karakteristik:
  → Timeframe: 2–5 hari, bisa sampai 1–2 minggu
  → Goal: exploit swings untuk pips lebih besar dari intraday
  → Fleksibilitas: antara intraday dan position trading

Keunggulan:
  → Less screen time vs day trading
  → Suitable untuk yang punya komitmen lain
  → Bisa trade saat news events (leverage untuk swing moves)
```

---

## 2 Model Swing Trading

```
Classic Swing Model (CSM):
  → Target: monthly range expansion
  → Durasi: 5 hari – beberapa minggu
  → Frekuensi: 1 trade per minggu
  → HTF: Monthly + Weekly
  → LTF: 4H

One Shot One Kill (OSOK):
  → Target: weekly range expansion
  → Durasi: 2–5 hari
  → Frekuensi: 2 trades per minggu
  → HTF: Weekly + Daily
  → LTF: 1H
```

---

## Main Premise Swing Trading

```
1. Weekly Range Expansion:
   → Identifikasi arah paling probable weekly candle akan expand

2. Day of Week:
   → Ekspektasi anchor point terbentuk Mon, Tue, atau Wed
   → Ini adalah basis untuk frame setup

3. Turtle Soup & Orderblock:
   → Tunggu orderblock terbentuk setelah turtle soup
   → Frame entry berdasarkan entry model
```

---

## Mechanics (Routine)

```
Preparation:

# Ep.2 — Kiss of Death (KOD)

  → Harga sudah membuat setup CRT

KOD         = Last refuel sebelum the big journey
Tanpa KOD   = Harga choppy / tidak clean

# Entry Models — CRT Enigma

## Model #1: The Thick Candle Break

```
Menangkap momen ketika false breakout gagal dan true reversal dimulai.

Untuk Short Entry:
  1. Harga break above old HIGH → thick strong-bodied bullish candle muncul
  2. Next candle closes BACK BELOW thick candle's body → breakout gagal
  3. Entry short on the close, SL di atas thick candle's high
  4. Target next support / CRT Expansion Level

Untuk Long Entry:
  1. Harga sweeps below old LOW → thick bearish candle
  2. Harga closes BACK ABOVE thick candle's body → purge complete
  3. Entry long on the close, SL di bawah thick candle's low
  4. Target next resistance / CRT EL

Note: Thick candle = deliberate institutional trap. Saat reject → kamu masuk
      di exact moment smart money reverses move yang mereka engineered.
```

## Model #2: CRT OTE (Optimal Trade Entry)

```
Continuation model — menangkap resumption momentum setelah retracement sehat.

Steps:
  1. Tunggu strong impulsive move yang breaks market structure
  2. Mark range dari low ke high (bullish) atau high ke low (bearish)
  3. Tunggu harga retrace 50-88.6% dari move tersebut
     → Idealnya ke FVG atau OB dekat candle EQ
  4. Entry direction of original impulse, SL beyond retracement extreme
  5. Target next expansion level atau liquidity zone

Logic: Setelah setiap drive kuat, market cari balance sebelum deliver next leg.
       Entry di titik balance ini, bukan saat exhaustion.
```

## Model #3: Kiss of Death (KOD) Turtle Soup

```
Move paling deceptive — second, smaller liquidity grab sebelum true expansion.

Steps:
  1. Setelah initial purge + partial mitigation, harga buat ONE MORE final test of purge level
  2. "Kiss" dari sisi lain = final trap untuk late traders
  3. Saat retest gagal dan harga reject kuat → entry direction yang intended
  4. SL just beyond the kiss point
  5. Target full Expansion Level untuk max reward

Warning: Advanced pattern, butuh precise timing dan confirmation.
         Practice extensively di demo sebelum live.

Perbandingan Model:
  Model #1  = Reversal traders         → Risk: Medium
  CRT OTE   = Continuation traders     → Risk: Low
  KOD       = Advanced precision       → Risk: High
```

# Turtle Soup + HTF Flow (Secrets of Time)

## Central Truth

```
"If you ignore the higher timeframe direction, your turtle-soup setups will often fail."

→ Trading LTF tanpa check HTF = trading distribution, bukan manipulation
→ Itulah kenapa copy trades tanpa context sering gagal — tidak aligned
→ HTF yang dimaksud: Weekly atau Monthly (tempat large players act)

Rule: Hindari countertrend trades KECUALI ada clear HTF reason
```

## 4-Step Learning Curve

```
1. Receive the information  → Pelajari idenya
2. Witness it work on charts → Lihat live di chart
3. Trade what you witnessed  → Execute
4. Fix mistakes              → Improve risk, entries, psychology

→ Focus lebih pada correcting mistakes daripada cari "holy grail"
→ Fixing execution = wins trades
```

## Turtle Soup Definition

```
Turtle Soup = trap retail stop-hunters dengan fake breakout → lalu reverse

Rules:
  → Cari false break dari obvious lows/highs
  → Trade reversal ketika harga close KEMBALI ke dalam range
  → Works best ALIGNED dengan HTF order flow, bukan against it
```

## Key Price Action Elements

```
Order Block (OB):
  → Candle/cluster di mana smart money placed orders
  → Diidentifikasi dari strong move away
  → Bullish OB = last bearish candle sebelum rally
  → Bearish OB = last bullish candle sebelum drop

Rejection Block:
  → Zone dengan large wicks yang menunjukkan rejection
  → Signal stop-hunting activity

FVG (Fair Value Gap):
  → Gap antara candles yang sering diisi kemudian
  → Tip: Jangan trade bullish FVG di premium zone — only buy in DISCOUNT zones

Close vs Wick:
  → Close above/below = LEBIH KUAT daripada wick
  → Wick through block + quick rejection = liquidity grab
  → Bodies yang tidak break = inefficiency fill expected

Homework: Remove wicks di charting tool → observe structure lebih jelas
          (FVGs dan Order Blocks lebih visible melalui body structure)
```

## Failed Order Block → Breaker Block

```
Failed Order Block:
  → Terjadi ketika bearish OB "ditembus" (harga close di atasnya)
  → Often leads to strong rally toward TOP of failed block
  → Bearish OB yang gagal = Bullish Breaker Block

Bearish version: Bullish OB ditembus ke bawah → Bearish Breaker
```

## Ranges, Accumulation & Distribution

```
Range = Accumulation
  → Harga bergerak antara range low dan high (20+ candles)
  → Smart money accumulates positions dalam range

Break above range high = Change of State (bullish)
  → Sampai higher timeframe key level berkata sebaliknya

Range → Expansion = classic A-M-D dalam timeframe yang lebih kecil
```

## Reading a Real Setup

```
1. Identify HTF bias (Weekly → Daily → H4)
2. Find key levels (opens, highs, lows)
3. Drop to execution TF (5m atau 1m). Cari:
   → Turtle soup / false break of low or high
   → Confluence dengan FVG, breaker, atau rejection wick
   → Confirmation: CLOSE KEMBALI DALAM range (body close, bukan wick)
4. Entry: setelah bullish/bearish confirmation close
5. Stop: di bawah swing low (invalidates idea)
6. Target: next HTF confluence (range midpoint, high, OB)
```

## Pre-Trade Checklist (Quick)

```
✓ Higher-TF bias sudah marked?
✓ Higher-TF key level(s) sudah marked?
✓ Turtle soup / false break visible?
✓ Close (body) confirms rejection?
✓ Logical stop dan target ada?

→ Jika jawaban ada yang "no" atau "I don't know" → SKIP TRADE
```

## Common Beginner Mistakes
