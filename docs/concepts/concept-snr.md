# Prinsip Dasar Malaysian SNR

## Filosofi Inti

```
Pasar bergerak karena MANIPULASI:
  → Setiap pelaku besar di pasar memanipulasi harga untuk keuntungannya sendiri
  → Mereka trade off liquidity (bulk orders) di setiap level S/R

Kebenaran fundamental:
  1. Jika harga akan bergerak NAIK → ia harus bergerak TURUN dulu
  2. Jika harga akan bergerak TURUN → ia harus bergerak NAIK dulu

"Liquidity will always exist and therefore so will manipulation." — Richard Wyckoff
```

## Perbedaan Malaysian SNR vs Konvensional

```
SNR Konvensional:  Menggambar S/R di HIGH dan LOW candle (including wicks)

Malaysian SNR:     Fokus HANYA pada BODY candle
                   → Close dan Open price
                   → Wicks DIABAIKAN dalam identifikasi level

Alasan:
  → Close/Open = keputusan final pasar per sesi
  → Wicks = zona manipulasi (di sinilah liquidity hunting terjadi)
  → Body = consensus price — zona yang paling dihormati market
```

# Identifikasi SNR Level

## Cara Menggambar SNR

```
Alat:  Gunakan LINE CHART terlebih dahulu untuk identifikasi
       Setelah level ditemukan, switch ke CANDLESTICK chart

RESISTANCE Level (A-Shape):
  → Tarik garis dari CLOSE candle BULLISH
  → Hubungkan ke OPEN candle BEARISH berikutnya
  → Pada line chart: terlihat seperti huruf "A"
  → Perpanjang garis ke kanan (extend right)

SUPPORT Level (V-Shape):
  → Tarik garis dari CLOSE candle BEARISH
  → Hubungkan ke OPEN candle BULLISH berikutnya
  → Pada line chart: terlihat seperti huruf "V"
  → Perpanjang garis ke kanan (extend right)

Di TradingView: gunakan "Extended Trendline"
  → Line → Extended Trendline
  → Settings: pilih "Middle Point" dan "Extend Right"
  → Titik #1 = Close candle pertama, Titik #3 = Open candle berikutnya
  → Titik #2 (Middle) = hanya untuk memindahkan seluruh garis
```

# Tiga Tipe SNR

## 1. Classical SNR

```
Definisi:
  → Harga menyentuh level SNR → BOUNCE / reversal
  → Level bertindak sebagai dinding: support menahan turun, resistance menahan naik

Cara trading:
  → Identifikasi level SNR
  → Tunggu harga menyentuh level dengan WICK (rejection candle)
  → Enter setelah konfirmasi rejection
```

## 2. Breakout (Flipped) SNR

```
Definisi:
  → Level SNR ditembus (breakout)
  → Setelah breakout, level BERBALIK FUNGSI:
    - Support yang ditembus → menjadi Resistance
    - Resistance yang ditembus → menjadi Support

Cara trading:
  → Identifikasi breakout dari SNR
  → Tunggu retest ke level yang sama (sekarang sudah flipped)
  → Enter di retest dengan konfirmasi rejection
```

## 3. GAP (Open-Close) SNR

```
Definisi:
  → GAP = zona antara close candle satu sesi dengan open candle sesi berikutnya
  → Pada HTF: GAP terlihat sebagai "Hidden Zone"
  → Pada LTF: HTF GAP = Breakout (Flipped) SNR

HTF GAP SNR = LTF Breakout SNR

Decision Level (H4 GAP):
  → Jika H4 Gap terbentuk pada saat menyentuh HTF SNR level
  → Ini adalah "Decision Level" → biasanya menghasilkan breakout untuk REVERSE trend
  → Buying setup: GAP di HTF + menyentuh HTF SNR dari bawah (bullish breakout)
  → Selling setup: GAP di HTF + menyentuh HTF SNR dari atas (bearish breakout)
```

# Fresh vs Unfresh SNR

```
Fresh SNR:
  → Level SNR yang BELUM PERNAH disentuh setelah terbentuk
  → Probabilitas bounce lebih tinggi
  → Likuiditas masih penuh di level tersebut

Unfresh SNR:
  → Level SNR yang sudah pernah disentuh/ditest sebelumnya
  → Probabilitas lebih rendah
  → Likuiditas sebagian sudah terambil

"Trade an SNR level that is fresh for higher probability."
```

# SNR + TOUCH

```
Rule Malaysian SNR untuk entry:
  → Perlu PRICE REJECTION (sentuhan wick candle) dari level SNR
  → Wick harus menyentuh level SNR
  → Ini adalah konfirmasi bahwa level dihormati market

Contoh:
  Bullish: Wick candle menyentuh support SNR dari atas → bounce → buy entry
  Bearish: Wick candle menyentuh resistance SNR dari bawah → rejection → sell entry

HTF Touch refinement:
  → Jika Daily rejection TIDAK memiliki Liquidity Sweep
  → Refinement ke H4 atau H1 untuk mengidentifikasi sweep
  → Daily → H4 → H1 refinement chain
```

# SNR + MISS

```
Definisi MISS:
  → SNR terbentuk → harga bergerak menjauh
  → Candle-candle berikutnya wicknya GAGAL menyentuh level SNR
  → Ini disebut "MISS"

MISS memvalidasi SNR level:
  → Semakin banyak candle yang "miss" level tersebut (tidak menyentuhnya)
  → Semakin kuat level SNR tersebut ketika akhirnya disentuh

Single Candle Breakout (aggressive):
  → Tipe gerakan yang menembus atau menyentuh SNR TANPA MISS terlebih dahulu
  → Lebih agresif dan berisiko untuk ditrading
  → Probabilitas lebih rendah karena tidak ada validasi MISS

"NO VALIDATION, NO TRADE!"
```

# Engulfing Patterns

## Definisi dan Tipe

```
Engulfing = candle kedua SEPENUHNYA menutupi candle pertama (termasuk body dan wick)

Fungsi:
  → Merepresentasikan PRICE REJECTION di pasar
  → Valid di semua instrumen dan semua timeframe

Tiga tipe Engulfing:
  1. Bullish Engulfing:   Candle bearish diikuti candle bullish yang lebih besar
  2. Bearish Engulfing:   Candle bullish diikuti candle bearish yang lebih besar
  3. Hidden Engulfing:    Engulfing yang tersembunyi di dalam HTF GAP SNR

Hidden Engulfing:
  → HTF GAP (contoh: H4) mengandung LTF Engulfing di dalamnya
  → Semakin turun ke LTF → Engulfing semakin jelas terlihat
  → H4 GAP SNR → direfine ke H1 → terlihat sebagai Engulfing (Flipped/Breakout SNR)
```

## Cara Menggunakan Engulfing

```
Engulfing Trading Steps:
  1. Identifikasi HTF SNR level (fresh atau unfresh)
  2. Tunggu harga mencapai level SNR
  3. Cari Engulfing pattern di level tersebut
  4. Konfirmasi dengan entry timing (session open)

Timing terbaik:
  → Market Opening Session
  → London Close
  → New York Open

Entry Confirmation:
  → Engulfing di SNR = konfirmasi
  → H4 GAP refined ke H1 → Engulfing terlihat = entry yang lebih detail
```

## GAP SNR Refined ke LTF Engulfing

```
H4 GAP SNR (Hidden Zone) = H1 Breakout Engulfing

Cara scalp dengan ini:
  1. Identifikasi H4 GAP SNR di HTF
  2. Refinement ke H1 → cari Engulfing
  3. Entry di LTF (H1 atau M30) berdasarkan Engulfing

"When HTF GAP has within it a LTF Engulfing, the Hidden Engulfing becomes more obvious 
when refined in LTFs. The lower you go, the more the Engulfing is made manifest."

Real example:
  H4 GAP SNR → H1 Engulfing → "Flipped Engulfing from chart refinement"
  Bonus: Retest setelah breakout + M15 rejection yang berinterseksi dengan H4 Breakout Trendline
```

# Trendline — Malaysian Approach

## Konsep Dasar

```
Trendline = ANGLED SNR
  → Trendline memetakan arah SNR dalam pasar yang trending
  → Bukan random — terhubung dengan SNR levels

Fungsi trendline:
  1. Menentukan uptrend, downtrend, atau sideways
  2. Menentukan titik entry

CATATAN: Trendline TIDAK berlaku untuk GAP SNR
(kecuali ketika GAP SNR direfine ke LTF Engulfing)
```

## Marriage Concept (Ariff T)

```
"Trendline harus dipasangkan dengan SNR level"
→ Ini disebut MARRIAGE CONCEPT oleh trader Malaysia Ariff T

Logic:
  → Trendline sendiri tidak valid tanpa SNR
  → SNR sendiri kurang presisi tanpa trendline
  → Ketika keduanya bersilangan di titik yang sama = konfluensi tertinggi

"VALIDITY atau SIGNIFICANCE = sesuatu yang mendapat lebih banyak perhatian dari yang lain"
→ SNR yang valid = SNR yang memiliki confluence dengan trendline
```

## Rules Penggunaan Trendline

```
7 Rules Trendline (Malaysian SNR):

1. Gunakan trendline untuk menghubungkan dari satu SNR ke SNR lainnya
2. Fokus pada tempat candle BODY close dan open — ABAIKAN wicks
3. Trendline TIDAK bisa menghubungkan GAP SnR
4. Pastikan minimal DUA SNR terhubung (upward atau downward)
5. TIDAK ada candle yang boleh CLOSE di atas/bawah titik #3 trendline
6. HANYA masuk trade ketika WICK candle menyentuh dan MENOLAK SNR di titik #3
7. JANGAN masuk trade di titik #2
   → Entry masih valid setelah titik #3 selama tidak ada candle close di luar trendline

Urutan titik:
  Titik #1 = SNR pertama
  Titik #2 = TIDAK untuk entry (terlalu dini)
  Titik #3 = Titik entry utama (confluence SNR + trendline)
```

## The "X" Factor Trendline

```
"X" Factor = ketika DUA trendline saling bersilangan
→ Silangan trendline = zona confluence ekstra
→ Price sering bereaksi kuat di titik persilangan dua trendline
→ Meningkatkan probabilitas entry secara signifikan

Penggunaan: Identifikasi crossover point → tunggu rejection di sana
```

# QML (Quasimodo Level) / HNS + Trendline

```
QML = Head and Shoulders pattern yang digunakan sebagai SNR

Kombinasi QML + Trendline:
  → QML memberikan kualitas setup yang lebih tinggi
  → "Right shoulder" = titik retest setelah breakout
  → Entry di right shoulder yang berinterseksi dengan trendline

Real example:
  LTF (M5): HNS SNR under HTF Engulfing zone
  Head menyentuh HTF SNR = great confluence
```

# Storyline dan Multi-Timeframe (MTF) Analysis

## Hierarki Timeframe

```
Monthly:  Tidak konsisten untuk menentukan direction
          → Digunakan sebagai context saja

Weekly:   MAIN DIRECTION (arah utama)
          → Tentukan apakah market bullish atau bearish dari sini

Daily:    Retracement / Roadblocks untuk arah Weekly
          → Di mana price kemungkinan pullback atau tertahan

H4:       Confirmation / Roadblocks (Gap level) untuk Daily dan Weekly
          → Identifikasi GAP SNR yang menjadi decision level

H1:       SANGAT SPESIAL
          → Price memutuskan apakah direction valid atau tidak di sini
          → HARUS memberikan wick candle atau gap candle untuk konfirmasi

Intraday: H4, H1, M30, M15, M5 — untuk entry dan fine-tuning
```

## Cara Membaca Storyline

```
Storyline berjalan dari:
  Monthly → Monthly (satu cycle bulanan)
  Weekly  → Weekly  (satu cycle mingguan)
  Daily   → Daily   (satu cycle harian)
  Kemudian intraday: H4, H1, M30, M15, dll.

Price bergerak:
  Support → Resistance → Support → Resistance (terus berulang)
  "Each timeframe has its own structure and which must be fulfilled."
```

# Roadblocks

```
Definisi:
  → Internal structural highs and lows di setiap timeframe
  → Area di mana price kemungkinan TERTAHAN sebelum melanjutkan

Kapan wajib diperhatikan:
  → Sebelum entry, periksa apakah ada roadblock antara entry dan target
  → Jika ada roadblock signifikan → probabilitas target tercapai lebih rendah

"It is highly essential for trader to watch out for roadblocks/obstacles 
(internal structural highs and lows) in every timeframe."

Roadblocks bisa berupa:
  → SNR level di timeframe yang lebih tinggi
  → GAP yang belum tertutup
  → Swing high/low dari candle sebelumnya
```

# 2 TF Confirmation Rule

```
Aturan konfirmasi dua timeframe:

Step 1: Identifikasi FRESH SNR di HTF
Step 2: Tunggu price menyentuh HTF SNR dengan rejection (wick)
Step 3: Turun DUA timeframe lebih rendah
Step 4: Cari breakout di LTF (SNR tidak harus fresh di LTF)

Pasangan timeframe:
  Weekly Setup   →  H4 Confirmation
  Daily Setup    →  H1 Confirmation

Setelah konfirmasi (LTF Breakout):
  → TUNGGU pullback untuk entry
  → Buy atau sell dari "right shoulder" atau QML
  → Right shoulder = retest ke peak (head) → entry di sana

Monitoring dua TF sekaligus:
  Daily SNR   → monitor Daily + H4
  H4 SNR      → monitor H4 + H1/M30
```

# Konflik Antar Timeframe

```
Jika Weekly BULLISH dan Daily BEARISH:
  → Weekly storyline TIDAK BISA melanjutkan bullish
  → Sampai Daily bearish storyline selesai terlebih dahulu
  → Artinya: Daily harus selesaikan bearish move-nya dulu

Logic:
  HTF selalu mendominasi LTF dalam jangka panjang
  Tapi LTF memiliki "job"-nya sendiri yang harus diselesaikan
  → Sabar menunggu LTF menyelesaikan strukturnya sebelum HTF bisa lanjut
```

# Change of Storyline

```
Perubahan Storyline = ketika trend arah berubah di suatu timeframe

Cara identifikasi:
  → Perhatikan kapan price tidak lagi bereaksi terhadap SNR dalam arah yang sama
  → Cari pola dimana price mulai membentuk higher low (bullish shift) 
    atau lower high (bearish shift) secara konsisten

Contoh:
  Sebelumnya: price bounce dari resistance lalu turun (bearish)
  Perubahan:  price menembus resistance, retests (flip ke support), lanjut naik
  → Ini adalah Change of Storyline bullish

Konfirmasi:
  Gunakan H1 sebagai "decision maker" — harus memberikan wick/gap candle
  yang mengkonfirmasi bahwa storyline sudah berubah
```

London Close:
  → Volatility berkurang → reversal sering terjadi
  → Gap antara London Close dan NY sering menjadi NWOG-style level

New York Open:
  → 9:30 AM NY = high volatility + institutional activity

  → High impact news = momentum besar
  → Tunggu initial spike selesai sebelum mencari SNR konfirmasi

# Konfluensi Terbaik dalam MSNR

```
Urutan kekuatan konfluensi (paling kuat ke paling lemah):

1. Fresh HTF SNR + Trendline (Marriage Concept) + Engulfing di sesi utama
2. HTF GAP SNR refined ke LTF Engulfing (Hidden Engulfing)
3. QML/HNS + Trendline + SNR intersection
4. Two Trendlines Crossing (X Factor) + SNR
5. Classical SNR + MISS validasi + session timing
6. Unfresh SNR + Trendline

"WHEN TWO PATTERNS COLLIDE — It confirms momentum!"
→ Dua pattern yang konfluens = konfirmasi momentum yang lebih kuat
```

# Workflow Malaysian SNR

```
Step 1: Tentukan Weekly Direction
  → Weekly = arah utama (bullish atau bearish)
  → Identifikasi Weekly SNR levels (V-shape atau A-shape)

Step 2: Analisis Daily Roadblocks
  → Apakah ada Daily SNR yang bisa menghambat Weekly direction?
  → Jika ada → tunggu Daily structure selesai dulu

Step 3: Tandai H4 GAP SNR (Decision Levels)
  → H4 GAP di dekat HTF SNR = potential decision level
  → Ini adalah area "explosive" untuk breakout dan reversal

Step 4: Monitor H1 untuk Konfirmasi
  → H1 = timeframe kritis dimana price memutuskan arah
  → Harus ada wick candle atau gap candle di H1

Step 5: Refinement dan Entry
  → Refinement: turun ke LTF untuk cari Engulfing atau rejection
  → Terapkan 2 TF Confirmation Rule
  → Entry di right shoulder atau QML setelah pullback

Step 6: Validasi dengan MISS
  → Pastikan ada MISS (beberapa candle yang tidak menyentuh level)
  → MISS = validasi kekuatan SNR level

Step 7: Target dan SL
  → Target: SNR berikutnya di arah trade (S ke R atau R ke S)
  → Stop Loss: di atas/bawah wick yang sudah rejected SNR
  → "NO VALIDATION, NO TRADE!"
```
